## ✨ Core Features | 核心特性

| Feature | Description |
|:---|:---|
| **🖥️ Zero-Dependency Architecture** | 纯原生 HTML5 / CSS3 / ES6 构建，零第三方 UI 库，页面加载速度 **< 50ms** |
| **🌌 Interactive Particle Network** | 全屏 Canvas 粒子网络，支持鼠标引力吸附交互，模拟「流量网络监控」概念 |
| **🃏 Glassmorphism 3D Cards** | CSS `backdrop-filter` 毛玻璃质感 + 原生 JS 实现的 3D Tilt 倾斜特效 + 动态高光 |
| **⚡ Cyber Glitch Typography** | `@keyframes` 驱动的红蓝通道分离 Glitch 故障动画，赛博朋克字体渲染 |
| **🔊 BGM Audio Payload** | 内嵌沉浸式音轨载荷，终端风格控制面板一键切换 |
| **📱 Fully Responsive** | CSS Grid `auto-fit` 自适应布局，从 4K 大屏到移动端完美适配 |
| **🧩 Config-Driven UI** | 数据驱动 DOM 架构，新增项目只需在 JSON 数组追加一个对象 |

## 🛠️ Tech Stack | 技术栈

```
 HTML5          ▸  语义化标签 / Canvas API / @font-face
 CSS3           ▸  Grid / Flexbox / Glassmorphism / @keyframes
 JavaScript     ▸  ES6+ / 粒子物理引擎 / 事件委托 / Clipboard API
 Fonts          ▸  Orbitron (Google Fonts) / Cyberpunk (Local)
 Deployment     ▸  GitHub Pages (Custom Domain)
```

**零依赖清单：**
- ❌ No React / Vue / Angular
- ❌ No Tailwind / Bootstrap
- ❌ No Three.js / GSAP / jQuery
- ✅ 100% handcrafted vanilla code

## 📁 Project Structure | 目录结构

```

│
├── index.html                 # 主页面 — 所有 HTML/CSS/JS 集于一身

├── bgm.mp3                    # 沉浸式 BGM 音轨

│
└── (未来扩展)
    └── 项目数据源 JSON 数组内置于 index.html

```
